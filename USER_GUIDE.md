## ğŸ“ Content Types

### Blog
- **Input**: Titel/Onderwerp, Doelgroep, Regio's, Taal
- **Output**: Complete blog artikel (1500-2200 woorden) met:
  - SEO metadata (title, description, keywords)
  - Gestructureerde secties
  - FAQ's
  - Interne links
  - CTA

### Product
- **Input**: Productnaam, Categorie, Use case, Doelgroep, etc.
- **Output**: Product pagina met:
  - SEO titel en meta description
  - H1 en intro
  - Beschrijving en voordelen
  - Use cases
  - CTA

### Landing
- **Input**: Onderwerp, Doelgroep, Tone of voice, etc.
- **Output**: Landing pagina met:
  - H1 en intro
  - Body blocks
  - FAQ's
  - Scenario's
  - CTA

### Categorie
- **Input**: Categorie naam, Beschrijving, etc.
- **Output**: Categorie pagina met:
  - Intro
  - Voordelen
  - Gebruikssituaties
  - Advies

### Social Media
- **Input**: Platform (Instagram/LinkedIn), Onderwerp, Afbeelding (optioneel)
- **Output**: Social media post met:
  - Platform-specifieke content (Carousel, Reel, LinkedIn Post)
  - Hashtags (8-15 voor Instagram, 3-5 voor LinkedIn)
  - CTA
  - Hook (sterke eerste zin)

- Elke keer dat je content opslaat, wordt een nieuwe versie gemaakt
- Je kunt eerdere versies bekijken en vergelijken

## ğŸ¨ Custom Prompts per Tenant

### Waarom custom prompts?

Niet elke klant wil dezelfde schrijfstijl. Bijvoorbeeld:
- **Broers Verhuur**: Praktisch, servicegericht, regio's (Haarlem, Amsterdam)
- **Andere klant**: Formeler, zakelijk, andere regio's

### Prompts aanpassen

1. Ga naar **Settings â†’ Prompts**
2. Selecteer de tenant in de dropdown
3. Pas aan:
   - **Base Instruction**: Algemene stijl (tone of voice, merknaam, etc.)
   - **Templates**: Per content type (blog, product, etc.)
4. Klik op **"Opslaan"**

### Prompts structuur

**Base Instruction:**
- Algemene schrijfstijl
- Tone of voice
- Merknaam en entiteiten
- Algemene regels

**Templates:**
- Specifieke instructies per content type
- JSON output structuur
- Verplichte velden
- Validatie regels

## ğŸ†˜ Veelgestelde Vragen

### Hoe maak ik een API key aan?
Zie `API_KEYS_GUIDE.md` voor gedetailleerde instructies per provider.

### Kan ik meerdere tenants hebben?
Ja! Maak zoveel tenants aan als je wilt. Elke tenant heeft zijn eigen configuratie.

### Werken prompts per tenant?
Ja! Als je een tenant selecteert, worden automatisch de prompts van die tenant gebruikt. Als die niet bestaan, worden de global prompts gebruikt.

### Kan ik content exporteren?
Ja! Op de result pagina kun je:
- Download als JSON
- Download als Markdown
- Export naar WordPress
- Export naar Webflow

### Hoe beheer ik gebruikers?
Alleen admins kunnen gebruikers beheren:
1. Ga naar **Settings â†’ User Management**
2. Klik op **"Gebruiker toevoegen"**
3. Vul email, naam, wachtwoord en rol in
4. Klik op **"Toevoegen"**

### Hoe koppel ik Instagram/LinkedIn?
1. Ga naar **Settings â†’ Social Accounts**
2. Klik op **"Connect Instagram"** of **"Connect LinkedIn"**
3. Volg de instructies
4. Zie `SOCIAL_MEDIA_SETUP.md` voor gedetailleerde setup

### Wat is Fast Mode?
Fast Mode is een optie voor snellere content generatie:
- Gebruikt snellere AI modellen
- Lagere temperature voor deterministische output
- Minder tokens voor snellere response
- Ideaal voor snelle iteraties

### Hoe werkt automatische posting?
1. Koppel je Instagram/LinkedIn account
2. Maak een post aan en plan deze voor later
3. De scheduler service post automatisch op de geplande tijd
4. Zie `SCHEDULER_SETUP.md` voor setup instructies
